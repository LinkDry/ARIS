## 三、核心发现

### 3.1 假设被证伪

> **原始假设**: 语义感知能提升资源分配效率
> **实验结论**: 在当前设定下，语义感知反而降低性能

### 3.2 可能原因

| 假设 | 解释 |
|------|------|
| 语义噪声 | LLM 嵌入引入无关信息，干扰决策 |
| 信息冗余 | 优先级已包含足够判别信息，语义是冗余 |
| 模型复杂度 | LLM-SRAF 过拟合，Priority-only 更简洁 |
| 奖励设计 | 当前奖励函数更适合纯优先级方法 |

### 3.3 边界条件

语义感知可能有效的场景:
- 用户请求高度个性化
- 业务类型差异巨大
- 需要跨域知识迁移
