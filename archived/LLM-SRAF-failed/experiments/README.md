# LLM-SRAF: LLM-Enhanced Semantic Resource Allocation Framework

大语言模型增强的卫星网络语义资源分配框架

## 项目概述

LLM-SRAF 是一个创新的卫星网络资源管理框架，利用大语言模型(LLM)的语义理解能力，将用户自然语言描述的业务需求转化为卫星网络资源分配策略。

### 核心特性

- **语义理解**: 利用 LLM 理解业务描述，提取资源需求
- **跨模态融合**: 融合语义、网络状态、卫星轨道等多模态信息
- **智能决策**: 基于 PPO 强化学习的资源分配决策
- **实时响应**: 端到端推理时延 < 100ms

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 生成数据

```bash
python -m src.data
```

### 3. 训练模型

```bash
# 训练语义理解模块
python scripts/train.py --mode semantic --epochs 10

# 训练 RL 模块
python scripts/train.py --mode rl --num_episodes 1000

# 完整训练
python scripts/train.py --mode all
```

### 4. 运行测试

```bash
pytest tests/test_model.py -v
```

## 模型架构

```
用户业务描述 ──┐
              │
网络状态信息 ──┼──> 跨模态融合 ──> 资源决策 ──> 分配策略
              │
卫星轨道信息 ──┘
```

### 核心模块

| 模块 | 功能 | 实现 |
|------|------|------|
| SemanticUnderstandingModule | 语义理解 | LLM编码器 + 资源映射 |
| CrossModalFusion | 跨模态融合 | 门控融合机制 |
| ResourceDecisionModule | 资源决策 | Actor-Critic + PPO |

## GitHub 同步

代码自动同步到 [GitHub 仓库](https://github.com/LinkDry/ARIS)。

```bash
# 初始化同步
bash scripts/sync_github.sh init

# 单次同步
bash scripts/sync_github.sh once "commit message"

# 定时同步 (每30分钟)
SYNC_INTERVAL=30 bash scripts/sync_github.sh start
```

### 同步规则

✅ **同步内容**:
- `experiments/configs/`
- `experiments/scripts/`
- `experiments/src/`
- `experiments/tests/`
